Description:

Commands (As Of Latest Release):


Utility And Moderation:


Ping: Shows ping latency. Usage: -ping


Purge: Deletes args ammount of commands. Usage: -purge <number>


Prefix: Changes bot prefix. Usage: -prefix <prefix>


PFP: Grabs given users profile picture. Usage: -pfp <@member>


Poll: Generates voting poll. Usage: -poll <message>


Kick: Kicks given member. Usage: -kick <@member>


Ban: Bans given member. Usage: -ban <@member>


Unban: Unbans given member. Usage: -unban <@member>


Help: Displays and explains commands. Usage: -help


Anime Reactions:


Hug: Hugs given user. Usage: -hug <@member>


Kiss: Kisses given user. Usage: -kiss <@member>


Poke: Pokes given user. Usage: -poke <@member>


Pat: Pats given user. Usage: -pat <@member>


Slap: Slaps given user. Usage: -slap <@member>


Fun:


Meme: Sends a meme from reddit. Usage: -meme


Reddit: Gets random post from given subreddit. Usage: -reddit


Pixabay: Gets free image using pixabay.com's api. Usage: -pixabay <query>


Joke: Responds with joke. Usage: -joke


Urban: Checks Urban Dictionary for definitions. Usage: -urban <query>


Dog: Returns image of dog. Usage: -dog


Cat: Returns image of cat. Usage: -cat
